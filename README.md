# Travel Date Picker

A Persian (Jalali) date picker component built with Nuxt 3.

## Setup

Install dependencies:

```bash
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```

Locally preview production build:

```bash
yarn preview
```
